## About the repo  

The main purpose of this work is to document all my learning through this Bioinformatics platform. The organization of the documents will be:  

-requirements.txt : this text file contains packages necessary for replicating some code (if needed).  
-src/ : this folder contains all the platfom exercises (all the problems that I am capable of solving) inside subfolders, following the very own platform problem organization.   
-data/ : this folder contains all the inputs and outputs tested in the platform, organized in subfolders.  
-docs/ if I feel the need to point out some important detail of any exercise, I will make it here in a Markdown file
